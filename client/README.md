This folder contains the frontend of **Prompt2Vdo**, a simple AI text-to-video prototype built using React.

The frontend allows users to:
- Enter a text prompt
- Select a video style and duration
- Send the request to the backend
- View the generated response or placeholder output

---

## Tech Stack
- React (Create React App)
- JavaScript
- CSS

---

## How to Run the Frontend

Make sure Node.js is installed.

```bash
cd client
npm install
npm start

```
The app will run on: 
http://localhost:3000
